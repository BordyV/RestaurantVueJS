Ajouter un restaurant à la base de données

Attention cette opération permet d'ajouter un restaurant avec un nom et une cuisine.

Cette fonctionnalité ne permet pas encore de pouvoir ajouter le quartier, le menu, l'addresse du restaurant.

Ces champs seront disponibles dans une version v2 de cette application.

Le nom n'est pas valide il doit être supérieur à 2 caractères La cuisine n'est pas valide, elle doit être supérieur à 2 caractères RESET Ajouter
